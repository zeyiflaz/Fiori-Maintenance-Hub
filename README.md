# Fiori Maintenance Hub
Bu proje, endüstriyel makinelerin bakım ve onarım süreçlerini dijitalleştirmek için geliştirilmektedir.

### 🛠 Teknolojiler
- **Frontend:** SAP Fiori / UI5
- **Database & Auth:** Supabase
- **Backend:** Node.js (veya kullandığın dil)

### 🚀 Mevcut Durum
- Proje geliştirme aşamasındadır.
- Veritabanı şeması ve statü bazlı yetkilendirme üzerinde çalışılmaktadır.
